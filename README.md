# Task Tracker

A simple **To-Do List application** built with **Blazor and C#**.

## Features

- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Filter tasks:
  - All
  - Completed
  - Active
- Display active tasks count
- Validation for empty input

## Task Properties

Each task contains:

- Title
- Start date
- Due date
- Completed status

## Technologies

- C#
- Blazor
- .NET
- Razor Components
- Bootstrap / CSS

## Data Storage

The application uses an **in-memory list** without a database.
